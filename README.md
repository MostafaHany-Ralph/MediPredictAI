## 🚀 Overview

**MediPredictAI** is an AI-powered healthcare prediction system designed to assist in early medical risk detection.  
The system analyzes patient data to:

- 🦴 Detect potential bone fractures  
- 🩸 Predict risk of Diabetes  
- ❤️ Assess probability of Heart Disease  
- 🧠 Estimate risk of Stroke or cardiovascular complications  

The goal of the project is to demonstrate how machine learning can be used to support early medical decision-making and improve preventive healthcare systems.

---

## 🧠 Key Features

🦴 Fracture Detection  
- Predicts possible bone fractures based on input medical indicators  

🩸 Diabetes Prediction  
- Analyzes patient data to estimate diabetes risk level  

❤️ Heart Disease Prediction  
- Evaluates cardiovascular health risk using trained models  

🧠 Stroke Risk Prediction  
- Identifies probability of stroke or brain-related complications  

⚙️ Unified AI System  
- Multiple models integrated into a single pipeline  
- Consistent prediction interface  

📊 Data-Driven Insights  
- Uses structured medical datasets for training and evaluation
